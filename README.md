<h2 align="center">KIR Jupyter Helper</h2>

<p align="center">
  <img src="./img/kir_jupyter_helper_logo.png" alt="logo" width="300"/>
</p>

kir-jupyter-helper is a command-line tool for registering custom Jupyter kernels on the BMRC cluster. 
It generates a wrapper script that handles module loading, and supports kernels backed by EasyBuild modules, conda environments, virtual environments, or Apptainer containers. Kernels can be registered privately for individual users or shared with a group via a shared project directory.
